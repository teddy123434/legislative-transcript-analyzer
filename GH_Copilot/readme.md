公督盟 委員會逐條審查文本委員發言次數與字數整理程式

# 結果輸出：Excel統計出該委員的發言次數與字數

1. 指定欲計算的委員
2. 用逐字稿頭銜計算發言次數
3. 刪除頭銜至冒號的部份，計算字數（記入標點符號） 
4. 輸出發言次數與發言字數

主席應最後計算，如果可以，給出各自委員說的話。

舉例：
